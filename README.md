# Agent
~~Text~~ hooker with javascript.

## Features
- Based on script.
- Hook everything.

### Folder Structure
```
agent.exe               <-- GUI
-data
  ├─scripts             <-- user scripts
  └─user-data           <-- Electron cache
-...
```
